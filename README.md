# 🧐 Sobre <a name="sobre"></a>

Este projeto foi desenvolvido como atividade avaliativa da disciplina de Programação Estruturada. O objetivo é aplicar na prática os conceitos estudados em sala de aula, como estruturas condicionais, estruturas de repetição, funções, vetores, matrizes e organização lógica do código.

O sistema foi construído em grupo, promovendo trabalho em equipe, divisão de tarefas e desenvolvimento colaborativo.

# 🎯 Objetivo <a name="objetivo"></a>

Desenvolver uma aplicação utilizando os conceitos de Programação Estruturada, criando um sistema funcional, organizado e de fácil utilização.

# ▶️ Como Executar <a name="como-executar"></a>

### Pré-requisitos

- Compilador C instalado
- Editor de código (VS Code, CodeBlocks, Dev-C++, etc.)

### Executando o projeto

1. Clone o repositório:

```bash
  git clone LINK_DO_REPOSITORIO
```

2. Acesse a pagina do projeto:
   
```bash
cd nome-do-projeto
```

3. Compile o codigo
   
```bash
gcc NOME_DO_ARQUIVO.c -o programa
```
4. Rode o programa

```bash
./programa
```
